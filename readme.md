# Kubit

Kubit helps you manage multiple kubernetes configs on the same machine

# Usage

* Add an entry
```sh
$ kubit add development ~/Projects/dev/kubeconfig
```

* List all entries
```sh
$ kubit list
```

* Remove an entry
```sh
$ kubit rm development
```

* print usage with no opts or the `help` subcommand
```sh
$ kubit help
```
